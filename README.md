## Quick Summary

Branch Worklog Manager is a local-first skill for recording branch work as Markdown timeline nodes, managing branch-level release status, archiving released branches, deleting branch history, and operating everything from a local Web UI.

`branch-worklog-manager` 是一个本地优先的技能，用来把分支工作记录为 Markdown 时间轴节点，管理分支级上线状态，归档已上线分支，删除分支历史，并通过本地 Web UI 统一管理。
